# AGENTS.md

## Project
This is the Altus Luna website, built with Next.js, TypeScript, Tailwind CSS, and the App Router.

## Brand direction
Altus Luna is a premium but approachable creative/media brand for:
- website builds
- drone shooting / aerial filming
- video editing
- social media content creation
- visual content for garden companies and landscaping businesses
- real estate, property, and small local business websites

The website should feel:
- premium
- warm
- modern
- trustworthy
- human
- not fake
- not overly dark
- not generic AI-looking

## Design rules
- Use a clean, light design.
- Use soft contrast, elegant spacing, and strong photography/video sections.
- Avoid childish gradients.
- Avoid generic SaaS template style.
- Keep sections spacious and professional.
- Prioritise mobile-first responsive design.

## Technical rules
- Use TypeScript.
- Use the Next.js App Router.
- Use Tailwind CSS.
- Keep components reusable.
- Do not install unnecessary packages.
- Do not create overly complex architecture.
- Split repeated UI into components.
- Keep code readable and simple.

## Website structure
The website should include:
- Home
- Services
- Portfolio / Work
- About
- Contact

## Core components
Create reusable components where useful:
- Header
- Footer
- Hero
- ServicesSection
- ServiceCard
- PortfolioPreview
- CTASection
- ContactSection
- TestimonialCard

## SEO rules
- Use semantic HTML.
- Use proper heading order.
- Use clear page titles and metadata.
- Add useful alt text for images.
- Write natural human copy.

## Working method
Before large structural changes, explain the plan first.
After editing, check for errors.
Run the build or lint command where appropriate.
